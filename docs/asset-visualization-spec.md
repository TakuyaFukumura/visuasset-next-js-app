# 資産可視化アプリ 仕様書

## 概要

リポジトリ内のCSVファイルからデータを読み込み、資産推移を棒グラフおよび表で表示するアプリケーションの仕様を定義します。

---

## 画面仕様

### 棒グラフ

- 資産の年次推移を棒グラフで表示する
- 横軸：年（例：2020、2021、2022、…）
- 縦軸：資産額（単位：万円）
- 各年の棒グラフはジャンルごとに色分けした積み上げ棒グラフとする
  - 株式
  - 現預金
  - 暗号資産

### 表

- 画面下部に資産推移の一覧表を表示する
- 表の列構成：

| 年 | 株式（万円） | 現預金（万円） | 暗号資産（万円） | 合計（万円） |
|----|-------------|--------------|----------------|------------|

---

## データ仕様

### CSVファイルの配置場所

- リポジトリ内の `data/` ディレクトリに配置する
- ファイル名：`assets.csv`
- パス例：`data/assets.csv`

### CSVファイルのフォーマット

- 1行目はヘッダー行とする
- 文字コード：UTF-8
- 区切り文字：カンマ（`,`）

#### ヘッダー行

```
year,stocks,cash,crypto
```

| カラム名 | 説明           | 単位 |
|---------|---------------|------|
| year    | 年            | 年（例：2020） |
| stocks  | 株式の資産額   | 万円 |
| cash    | 現預金の資産額 | 万円 |
| crypto  | 暗号資産の資産額 | 万円 |

#### CSVサンプル

```csv
year,stocks,cash,crypto
2020,100,200,50
2021,150,220,80
2022,180,210,60
2023,200,250,90
2024,220,260,100
```

---

## データ読み込み仕様

- アプリケーション起動時またはページロード時に `data/assets.csv` を読み込む
- CSVのパースにはサーバーサイドの処理を使用する（Next.js の Server Components または API Route）
- データは年昇順で表示する

---

## 将来的な拡張

- 資産ポートフォリオ機能を追加する際は、CSVのジャンル列を追加することで対応可能とする
- 例：不動産（real_estate）、投資信託（mutual_fund）などの列を追加
